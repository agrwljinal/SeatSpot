#  SeatSpot – Stop Roaming, Start Coding

SeatSpot is an Android application designed to streamline seat booking for the IGDTUW computer centre. It enables student users to reserve seats efficiently using an interactive seat map, reducing manual conflicts and improving space utilization.

---

##  Features

- 📍 **Seat Booking System**  
  Book seats seamlessly with an intuitive interface  

- ⏱️ **Time Slot Selection**  
  Choose preferred time slots for reservations  

- 🪑 **Interactive Seat Map (9×6 Grid)**  
  Visual layout for easy seat selection

- 🌡️ **Heatmap Visualization**  
  Distinguishes between:
  - Vacant seats  
  - Reserved seats  
  - Occupied seats  

- 📲 **QR Code Generation**  
  Generate QR codes for verification at entry  

- 📖 **View My Bookings**  
  Track all your bookings in one place
